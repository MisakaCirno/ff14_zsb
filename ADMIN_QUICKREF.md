- **用户名**: admin

## 主要功能

### ✅ 已启用的管理模块
1. **战术板分享管理**
   - 查看/编辑/删除所有分享
   - 批量设为公开/私有
   - 按作者、时间、状态筛选
   - 搜索标题、ID、描述

2. **用户管理**
   - 查看所有用户和昵称
   - 内联编辑用户资料
   - 权限和组管理
   - 用户激活/停用

3. **用户资料管理**
   - 编辑昵称和简介
   - 查看分享统计
   - 按时间筛选

## 增强特性

### 🌟 界面优化
- ✅ 完全中文化界面

- ✅ 显示昵称而非用户名

### 🔍 搜索增强

### 📊 数据展示
- 列表分页（每页20条）
- 日期层级浏览
- 字段排序

### ⚡ 批量操作
- 批量设为公开/私有
- 批量删除（需确认）
- 自定义批量操作

## 快速操作

### 修改管理员密码
```
管理后台右上角 → 修改密码
```

### 创建新管理员
```bash
python manage.py createsuperuser
```

### 查看操作日志
```
管理首页 → 最近操作
```
